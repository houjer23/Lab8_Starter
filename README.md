# Lab8-Starter
Graceful degradation means building an app that works but with reduced functionality and ideal conditions aren't met. Service workers apply this idea to the network: they intercept requests and serve cached responses when the network is slow or offline, so the app keeps working instead of breaking outright. Graceful degradation is the principle; the service worker is the mechanism that delivers it for offline support.
